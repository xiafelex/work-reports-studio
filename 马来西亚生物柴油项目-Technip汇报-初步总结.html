马来西亚生物柴油项目与 Technip 协作汇报初步总结

一、项目背景

为推进马来西亚生物柴油项目与 Technip 的前期工作交流，结合易高项目既有执行经验，现就项目协作模式、主要物资组织方式、执行基础及下一步工作安排作汇报说明。

本稿用于现阶段工作交流，并作为后续校核主材分交表、品牌清单和责任矩阵的基础文件。

二、建议的协作模式

建议本项目按以下方式组织实施：

  • Technip 负责总体设计、ISBL 设计、关键工艺系统集成、关键设备技术接口及系统责任。
  • 华东院负责 OSBL 设计、公用工程及配套系统设计。
  • NCC 负责执行型采购组织、中国境内催交与监造、检验、发运、跨境物流实施、现场施工以及马来西亚本地审批配合。

按上述方式组织，Technip 负责关键工艺系统、关键设备接口及整体性能责任，NCC 负责中国供应链组织、制造协调、国内预制、跨境物流和现场实施工作。

三、NCC 已具备的执行基础

1. 中国主材和设备组织基础

结合易高项目，NCC 已组织过多个专业的采购执行工作，主要涉及以下类别：

  • 管道主材
  • 管件法兰
  • 阀门
  • 储罐及附件
  • 电缆
  • 桥架及配电设备
  • 仪表及弱电材料
  • 消防系统材料
  • 油漆、防腐及预制耗材

已使用的主要厂家和品牌包括：

  • 管道主材方面：浙江久立、江苏武进、中兴能源、杭州海泰、扬州管件、合肥实华、江苏顺通等。
  • 阀门方面：江苏大阀、凯斯通、安徽铜都等。
  • 储罐附件及计量系统方面：湖北天信、江苏泽拓等。
  • 电气设备方面：镇江西门子母线、金海新源、南京凯拓、安徽天康等。
  • 油漆、防腐及预制耗材方面：佐敦涂料、江苏瑞瓦、南京昌泰、南京柯威等。

上述资源可覆盖项目执行阶段的询价、评审、催交、监造、检验、出厂资料组织及交付协调工作。

2. 国内预制协同基础

NCC 已具备将预制边界前置纳入采购和制造组织的工作基础，主要包括：

  • 管材、管件、法兰等预制关联物资的同步组织。
  • 焊材、板材、油漆、防腐等预制配套物资的前置准备。
  • 结合预制厂能力，对分段尺寸、包装方式和出海条件进行协调。

相关做法已在易高项目中实际应用，可直接支撑管道预制、储罐预制及相关配套材料组织工作。

3. 马来西亚属地采购和现场适配基础

结合易高项目，NCC 已形成马来西亚属地采购和现场适配方面的工作基础，相关厂家和供货渠道包括：

  • 电缆方面：XIN DAO、MAGNUM CABLE、PTH SAFETY 等。
  • 消防方面：CRS 等。
  • 防火涂料方面：RUSHOE 等。
  • 弱电和监控方面：AI SECURITY、ONE GASMASTER 等。

上述资源可满足现场安装响应、本地规范适配和属地配套组织要求。

4. 物流和现场实施基础

NCC 已具备从国内制造端到马来西亚现场的完整实施经验，包括：

  • 工厂发运
  • 报关与港口操作
  • 海运 / 空运组织
  • 到港后清关及陆运到场
  • 大件设备运输、路线调查、道路加固及吊装窗口协调

在现场实施方面，NCC 已具备土建、钢结构、设备安装、工艺管道、电仪等多专业施工组织能力，并具备较成熟的焊接质量控制和全过程追溯能力。

5. 马来西亚本地审批配合基础

NCC 在马来西亚项目中，已具备围绕本地审批和投运接口开展配合工作的经验，重点包括：

  • Bomba certificate
  • CCC
  • JKKP / DOSH 相关审批
  • boiler、natural gas、jetty import / export pipeline 相关许可
  • SAJ、TNB 等投运接口工作

四、对项目分工的理解

1. 关键系统责任应保持清晰

直接影响工艺性能、系统完整性、整体考核和关键技术接口的内容，建议由承担系统责任的一方统一组织。

这类内容通常包括：

  • reformer
  • boiler
  • 关键 exchanger
  • rotating package
  • 关键工艺成套包

上述内容与整体性能、系统联保和关键接口关系紧密，宜由同一责任主体集中管理。

2. 中国供应链优势应充分发挥

中国供应链成熟、制造资源集中、需要持续催交和制造过程控制的设备与材料，宜由中国采购组织统一推进。

这类物资通常包括：

  • 不锈钢管、碳钢管
  • 管件、法兰
  • 工艺阀门
  • 储罐附件
  • 桥架
  • 低压柜、中压柜、变压器
  • 焊材、油漆、防腐材料

上述物资在中国制造基础、交付组织和过程控制方面更具优势。

3. 预制边界应前置确定

与国内预制、分段出运、包装方式和现场安装顺序密切相关的物资，建议在设计和采购阶段同步明确。

这类物资通常包括：

  • 预制管段相关管材和管件
  • 储罐型材及附件
  • 焊材、板材
  • 管道油漆和储罐油漆

上述内容在前期明确后，后续制造、发运和现场安装安排会更顺畅。

4. 属地适配内容应单独识别

明显受马来西亚本地规范、现场安装响应速度和审批流程影响的物资和工作内容，建议单独识别并纳入属地实施安排。

这类内容通常包括：

  • 高低压电缆
  • 仪表电缆、控制电缆
  • 消防泵
  • 防火涂料
  • 监控及弱电系统
  • 部分现场安装配套材料和施工机具

上述内容与现场条件、本地供货响应和审批配合关系较大，宜结合属地资源统筹安排。

五、围绕四个重点问题的说明

1. 关于主材分交和品牌

根据现有资料，主要物资类别、供应商资源基础以及执行方式已基本清晰。下一步工作重点放在主材分交表和品牌清单的联合校核。

现阶段可先围绕关键系统、主要设备类别、执行场景和供应商资源基础形成一致认识，再进一步落实正式分交表和品牌清单。

2. 关于运输与物流

海外项目物流需要与设计、制造、预制、报关、运输、吊装和现场条件统筹考虑。

建议前期同步关注以下事项：

  • 大件设备的运输可行性
  • 预制构件的分段与包装安排
  • 到港后清关及陆运路线条件
  • 需要本地审批和特殊监管的物资清单

3. 关于执行边界优化

边界划分建议围绕以下几个方面统一考虑：

  • 系统责任是否清晰
  • 采购执行是否高效
  • 现场交付是否可控
  • 资料和审批接口是否顺畅

后续讨论中，建议围绕关键系统、执行型主材、预制关联物资和属地适配内容逐项识别。

4. 关于品牌与替代策略

品牌讨论建议同时关注以下方面：

  • 是否满足关键技术要求
  • 是否具备稳定交付能力
  • 是否具备海外项目经验
  • 是否能够满足第三方检验和本地审批资料要求
  • 是否能够支持现场实施和后续运维

对于关键设备，可结合 Technip 技术要求与 NCC 已有供应商资源，进一步形成国际品牌和中国优质品牌的对应备选方案。

六、希望与 Technip 形成的工作共识

建议优先形成以下几方面共识：

  • 项目整体执行按联合工作模式组织推进。
  • 关键系统责任与执行型采购组织分别由最合适的团队承担。
  • 中国采购、国内预制、跨境物流和现场实施作为一个整体统筹安排。
  • 马来西亚属地采购和本地审批工作单独识别并纳入执行策划。
  • 双方尽快共同校核主材分交表、主要设备清单和品牌清单第一版。

七、下一步建议

建议双方在本次交流后，尽快推进以下工作：

  • 共同确认项目主要设备和材料清单。
  • 共同识别关键系统、关键长周期设备和现场适配型物资。
  • 校核主材分交原则和品牌策略。
  • 明确国内预制边界及物流约束条件。
  • 梳理马来西亚本地审批及属地采购重点事项。

八、结语

NCC 希望在本项目中，与 Technip 形成一套责任清晰、接口顺畅、执行高效的合作模式。

基于易高项目已验证的采购、预制、物流、施工和本地审批经验，NCC 可承担项目执行侧的组织和实施工作，并为项目整体交付提供稳定支撑。
