# 马来西亚生物柴油项目分工策划方案

## 一、总体分工原则

结合易高马来西亚可再生燃料项目执行经验，建议本项目采用“德西尼布总设计及 ISBL 设计、华东院 OSBL 设计、NCC 负责采购执行与施工落地”的总体模式推进。

总体分工建议如下：

- 德西尼布负责总设计、ISBL 设计、关键工艺系统集成、关键设备技术接口及系统责任。
- 华东院负责 OSBL 设计、公用工程系统设计、辅助设施设计及相关配套接口设计。
- NCC 负责乙供范围采购、土建及地管范围采购、甲定乙采范围采购，并负责中国境内供应商协调、催交、第三方监造、出厂检验、发运、到马来西亚的物流组织、现场施工及本地审批配合工作。
- 对关键系统完整性和性能责任影响较大的甲供物资，建议由德西尼布主导采购。

## 二、专业分工建议

### 1. Engineering 分工

- 德西尼布负责总体技术路线、总体设计、ISBL 设计、关键工艺单元设计、关键设备 datasheet 及系统性能责任。
- 华东院负责 OSBL 设计、公用工程系统、储运系统、总图及厂外配套接口设计。
- NCC 负责从采购、预制、运输、吊装、现场安装等执行角度提出设计前置配合意见，反馈设备分段、运输限界、预制边界及施工可实施性建议。

后续建议按以下方式继续细化：

- ISBL 按工艺单元划分，如 700、800、900 等单元。
- OSBL 按系统划分，如 hydrogen、plant air、水处理、natural gas、steam、fire water、tank farm、loading/unloading、jetty pipeline 等。

### 2. Procurement 分工

建议采购边界按“三类划分”明确：

- A 类：关键工艺成套设备及长周期核心设备，由德西尼布主导采购。
- B 类：技术标准需统一、但采购执行适合国内统筹推进的物资，采用甲定乙采。
- C 类：大宗材料、施工强接口物资及国内成熟供应链设备，由 NCC 主导采购。

建议由德西尼布主导采购的范围包括：

- 核心工艺成套设备；
- 长周期且责任边界敏感的关键设备；
- reformer、boiler、关键 exchanger、关键 rotating package 等对系统责任影响较大的设备。

建议由 NCC 主导采购的范围包括：

- 管道主材、管件、阀门、支吊架、紧固件；
- 钢结构附材、防腐保温材料、电仪散材；
- 土建、地管、给排水、消防常规材料；
- 业主未指定欧洲品牌、且中国供应链成熟的常规动静设备。

### 3. Construction 分工

NCC 负责土建、地管、钢结构、设备安装、工艺管道、电仪施工、试车支持、保运及移交工作。

可依托易高项目经验重点体现以下执行能力：

- 中国境内采购与预制、海外现场安装的一体化执行能力；
- 大件设备运输及大型设备吊装组织能力；
- 多材质、高标准 ASME 体系焊接质量控制能力；
- 中方团队与属地团队协同实施能力。

### 4. Local Approval 配合

建议由 NCC 结合项目执行统筹推进本地审批和取证配合工作，包括但不限于：

- Bomba certificate；
- CCC；
- JKKP / DOSH 相关审批；
- jetty import/export pipeline 审批；
- boiler certificate；
- natural gas certificate；
- SAJ certificate；
- TNB certificate。

## 三、易高项目经验参考

易高项目已验证“中国境内采购与预制 + 海运 + 海外现场安装”的执行模式，具备以下参考意义：

- 中国境内制造、预制可充分发挥供应链、成本和制造效率优势；
- 海外现场安装可降低现场预制工作量，提高施工组织效率；
- 国内外团队联动，可实现采购、催交、监造、发运、清关、到场安装的全过程衔接；
- 本地审批、CCC、消防验收、DOSH/JKKP 等工作需与项目执行同步推进。

## 四、主材分交、品牌及运输建议

### 1. 主材分交建议

目前建议先按设备类别和责任逻辑梳理主材分交边界，后续结合易高项目主材分交表和最终品牌清单进一步细化。

初步建议如下：

- A 类：关键工艺成套设备及长周期核心设备，由德西尼布主导采购。
- B 类：技术标准需统一、但采购执行适合国内统筹的物资，采用甲定乙采。
- C 类：大宗材料、施工强接口物资及国内成熟供应链设备，由 NCC 主导采购。

### 2. 品牌建议

在品牌策略上，建议：

- 客户未指定欧洲品牌的设备，可优先采用中国成熟厂家。
- 关键设备采用“国际品牌备选 + 中国优质品牌备选”的双清单模式。
- 常规设备重点关注海外业绩、交期稳定性、第三方检验配合度及本地认证适配性。

NCC 可结合项目需要提供 reformer、boiler、exchanger、pumps 等设备的供应商长名单和推荐短名单，供技术比选使用。

### 3. 运输建议

结合易高项目经验，国际物流建议按“国内段 + 国际段 + 马来西亚段”统筹组织：

- 国内段：厂家发运、订舱、报关、港口操作；
- 国际段：海运 / 空运；
- 马来西亚段：换单、港杂、清关、陆运至现场。

运输方式建议如下：

- 大件、超限、长周期设备采用散杂船或重大件海运；
- 常规设备和材料采用集装箱海运；
- 紧急缺件和高时效性物资采用空运。

同时建议重点关注以下事项：

- 受 CIDB、DOSH 等管控的物资需提前准备进口准证及相关资料；
- 大件运输需提前完成 route survey、道路加固、清障和吊装窗口策划；
- 设计阶段应同步确定设备分段、运输尺寸和吊装方式，减少后续返工；
- 到港设备和材料需统一质保书、铭牌、材质证明、原产地及标签资料，确保顺利清关和报验。

## 五、后续工作建议

后续建议在现有分工框架基础上，进一步完善以下内容：

- 易高项目主材分交表；
- 最终品牌/厂家清单；
- 甲供、乙供、甲定乙采边界表；
- 重点设备供应商长名单及短名单；
- ISBL / OSBL 详细责任矩阵。
