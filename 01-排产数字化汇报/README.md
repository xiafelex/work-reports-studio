# 排产数字化汇报

这一层承接围绕“排产数字化主线”的汇报输出。

适合放这里的内容：

- 排产专题汇报文稿
- 进度逻辑分析
- 成本价值、机关赋能、施工组织与数字化结合的论证稿
- 与排产算法、排产方法、排产专题论证直接相关的材料

说明：

- `文稿与分析/` 放可直接写进汇报的排产主线分析稿
- [排产](/Users/felex/Documents/Codex/生物柴油模式/排产) 和 [排产PPT素材](/Users/felex/Documents/Codex/生物柴油模式/排产PPT素材) 也是这条主线的重要专题资产

如果这条主线产出的方法、框架或复盘具有长期复用价值，建议同步回 `ai-memory-vault`。

不放这里的内容：

- 项目进度跟踪资料
- 物流状态资料
- 图纸清单和现场基础资料
- 采购执行类资料

这些更适合进入 `02-项目介绍汇报/`，因为它们主要服务项目介绍和项目策划型汇报。
